# Anki 优质卡片制作原则

理解了 Anki 的核心原理，接下来看如何制作高质量卡片。这些原则将帮助你创建出高效、易于复习的卡片。

## 原则一：一次只问一个问题

一张卡片应该只包含一个知识点。问题越具体，复习越专注。

```
❌ 不好：单词：ubiquitous，意思是"无处不在"，发音是 [juːˈbɪkwɪtəs]

✅ 好：正面：ubiquitous 是什么意思？
     背面：无处不在的
```

**为什么要这样做？**
- 多个知识点会增加认知负担
- 无法准确判断对每个知识点的掌握程度
- 复习时间变长，但效果不一定更好

---

## 原则二：保持卡片简洁

卡片越简单，复习越高效。避免在一个卡片里塞太多信息。

**反面案例**：
```
正面：ubiquitous
背面：
- 词性：adj.
- 含义：无处不在的，普遍存在的
- 发音：[juːˈbɪkwɪtəs]
- 例句1：This technology has become ubiquitous.
- 例句2：Mobile phones have become ubiquitous in modern society.
```

**正确做法**：拆分成多张卡片
```
卡片1：ubiquitous → 无处不在的
卡片2：ubiquitous 发音 → [juːˈbɪkwɪtəs]
卡片3：例句填空 → This technology has become ___.
```

---

## 原则三：使用上下文

将单词放在例句中，比单独记忆单词效果更好。上下文帮助大脑建立更丰富的记忆连接。

```
❌ 不好：ephemeral = 短暂的

✅ 好：正面：The ephemeral nature of... 的中文是？
     背面：短暂的；瞬息的
```

**为什么上下文很重要？**
1. 帮助理解词的用法
2. 建立情境记忆，更容易回忆
3. 避免孤立记忆导致的"认识但不会用"

---

## 原则四：善用 Cloze 模板

完形填空（Cloze）模板非常适合学习短语、固定搭配和上下文关联。

### 基本语法

```
{{c1::答案}} 表示挖空部分
c1 是编号，相同编号的挖空会生成同一张卡片的不同视图
```

### 适用场景

**场景1：学习单词**
```
{{c1::Ubiquitous}} means existing or being everywhere.
```

**场景2：学习短语**
```
The {{c1::path}} to success is paved with hard work.
```

**场景3：学习固定搭配**
```
{{c1::Bear}} in mind that practice makes perfect.
```

### 进阶技巧

**多编号挖空**：
```
{{c1::Ephemeral}} means {{c2::temporary or short-lived}}.
```

这会生成两张卡片：
- 卡片1：Ephemeral means ___.
- 卡片2：___ means temporary or short-lived.

---

## 原则五：使用主动回忆式问答

避免被动识别，优先使用主动回忆。

```
❌ 被动识别：这个词是什么意思？ ephemeral

✅ 主动回忆：___ means temporary or short-lived.
```

主动回忆比被动识别的记忆效果好 **2-3 倍**。

---

## 原则六：添加图片和音频

多感官刺激可以显著提升记忆效果。

### 图片

适合：视觉学习者、需要记忆图像的场景

示例：
```
正面：[解剖学图片，标注肝脏位置]
背面：肝脏位于腹腔右上部，是人体最大的实质性器官
```

### 音频

适合：语言学习、需要记忆发音的场景

示例：
```
正面：ubiquitous 发音
背面：[播放按钮] [juːˈbɪkwɪtəs]
```

---

## 原则七：避免歧义

卡片的问题应该清晰、无歧义。

```
❌ 有歧义：这个词对吗？

✅ 清晰：ubiquitous 的中文意思是？
```

---

## 常见问题

### Q: 一张卡片可以同时有图片和文字吗？

A: 可以，但建议保持简洁。如果文字已经足够表达，就不需要添加图片。

### Q: 可以用中文问、英文答吗？

A: 可以，根据你的学习目标决定。如果是为了中译英，就用中文问；如果是为了英语思维，就用英文问。

### Q: 卡片太简单会不会没效果？

A: 不会。简单的卡片更容易坚持复习。关键是保持每日复习的习惯。

---

## 总结

| 原则 | 核心要点 |
|-----|---------|
| 一次一问 | 一个卡片，一个知识点 |
| 保持简洁 | 避免信息过载 |
| 使用上下文 | 例句比单独单词更好 |
| 善用 Cloze | 适合短语和固定搭配 |
| 主动回忆 | 闭眼回忆比看答案更好 |
| 多感官 | 图片和音频增强记忆 |
| 避免歧义 | 问题要清晰明确 |
