# 国内用户如何安装 Anki？2026完整指南

对于国内用户来说，安装 Anki 可能会遇到下载速度慢、访问官网困难等问题。本文将详细介绍几种在国内安装 Anki 的方法，帮助你快速上手这款强大的记忆工具。

## 一、认识 Anki 官网

Anki 官方网站为 [apps.ankiweb.net](https://apps.ankiweb.net/)。

**重要提醒**：国内存在多个仿冒 Anki 的应用（如 Anji记忆卡、Anymo 等），请认准官方橙色星星图标，下载地址一定是 `apps.ankiweb.net`。

## 二、电脑端安装（Windows/macOS）

### 方法1：使用官方下载器（推荐）

从 Anki 25.06 版本开始，官方不再提供完整安装包，而是提供下载器（Launcher）。

**安装步骤：**

1. 访问 [Anki 下载页面](https://apps.ankiweb.net/)，点击 "Download Anki"

2. 下载对应系统的启动器（Windows 选择 .exe，macOS 选择 .dmg）

3. 双击启动器，会打开一个终端窗口

4. 在终端中输入数字 `7`，回车打开 "Download mirror options"

5. 输入数字 `2`，开启中国区下载通道

6. 之后直接按回车键即可下载安装

```
菜单说明：
1) Latest Anki - 安装最新版
2) Choose a version - 选择指定版本
3) Keep existing version - 保持当前版本
4) Revert to previous version - 回退到前一个版本
5) Allow betas - 是否使用测试版
6) Cache downloads - 是否保存安装包
7) Download mirror - 下载镜像选项
8) Uninstall - 卸载软件
```

### 方法2：使用镜像站下载

部分国内镜像站提供 Anki 安装包下载，但需注意选择可信来源，避免下载到仿冒版本。

### 方法3：百度网盘备份

部分用户会在网盘分享 Anki 安装包，如需使用请确保来源可靠。

## 三、移动端安装

### Android 版本

**下载地址（按优先级排序）：**

1. Google Play（海外用户）
2. [GitHub releases](https://github.com/ankidroid/Anki-Android/releases)
3. 国内可信赖的镜像站

**特点**：Android 版完全免费，无广告

### iOS 版本

- **App Store**：[AnkiMobile Flashcards](https://apps.apple.com/zh/app/ankimobile-flashcards/id373493387)
- **价格**：$24.99（一次性付费）
- **开发者**：Damien Elmes（Anki 官方）

**注意**：App Store 上存在多个仿冒应用，请认准开发者为 "Damien Elmes"。

## 四、常见问题

### Q: 为什么下载速度很慢？

A: 国内访问 Anki 官方服务器较慢，建议使用上述方法开启中国区镜像下载。

### Q: iOS 版太贵了怎么办？

A: 可以选择 Android 版的 AnkiDroid 完全免费，功能与桌面端一致。

### Q: 如何判断下载的是不是正版 Anki？

A: 正版 Anki 图标为橙色星星，官网地址为 `apps.ankiweb.net`，桌面端下载的启动器名称为 "Anki-VERSION-setup.exe" 或类似格式。

### Q: Anki 需要联网才能用吗？

A: 不需要。Anki 支持离线使用，同步功能为可选项。

### Q: 国内有没有 Anki 中文网？

A: 存在名为 "Anki中文网" 的网站，但据官方论坛反馈，部分国内网站存在误导用户下载仿冒应用的情况。建议直接使用官方渠道或本文推荐的替代应用。

## 五、版本更新

Anki 目前最新稳定版为 **25.09.2**（截至 2026年6月）。

升级方法：打开桌面端 Anki → 菜单栏 "工具" → "升级/降级"

## 六、同步服务

### 官方同步

AnkiWeb 提供免费云同步服务，支持跨设备同步卡片库和进度。

### 自建同步服务器

有技术条件的用户可以在 NAS 或服务器上部署 Anki Sync Server，实现私有化同步。
