# Anki 基础卡片创建教程：从零开始制作你的第一张卡片

很多人听说 Anki 很强大，却不知道如何开始。创建卡片听起来复杂，其实只要几分钟就能上手。

今天，我将手把手教你创建第一张卡片。

## 一、前置准备

在开始之前，确保你已经：
1. 安装了 Anki 桌面端（参考：[Anki 国内安装指南](02-anki-install-guide.md)）
2. 了解了 Anki 的基本概念（参考：[Anki 核心概念详解](03-anki-core-concepts.md)）

## 二、认识卡片编辑器

打开 Anki，点击顶部的"添加"按钮（快捷键 `A`），你会看到卡片编辑器：

**界面布局（从左到右，从上到下）**：

1. **顶部**：类型下拉框（选择"基本"等模板）
2. **中间**：
   - 左侧文本框：正面（输入问题）
   - 右侧文本框：背面（输入答案）
3. **中下部**：牌组下拉框、标签输入框
4. **底部**：添加按钮

> 注：不同版本界面略有差异，但核心区域相同

### 界面说明

| 区域 | 说明 |
|-----|------|
| 类型 | 选择卡片模板（基本、Cloze等） |
| 正面 | 输入问题或提示 |
| 背面 | 输入答案 |
| 牌组 | 选择卡片存放的牌组 |
| 标签 | 为卡片添加分类标签（可选） |

## 三、创建你的第一张卡片

### 步骤1：选择或创建牌组

如果你是第一次使用，需要先创建一个牌组：

1. 点击"牌组"下拉框旁边的"管理牌组"
2. 点击"添加"
3. 输入牌组名称（如"英语单词"）
4. 点击"确定"

### 步骤2：输入正面内容

在"正面"区域输入问题：

```
ubiquitous 是什么意思？
```

### 步骤3：输入背面内容

在"背面"区域输入答案：

```
adj. 无处不在的；普遍存在的

例句：Mobile phones have become ubiquitous in modern society.
```

### 步骤4：添加标签（可选）

在"标签"区域输入分类名称，如：

```
英语 托福 词汇
```

多个标签用空格分隔。

### 步骤5：点击添加

点击右下角的"添加"按钮（或按 `Ctrl+Enter`），卡片就创建完成了！

## 四、基本卡片类型详解

基本卡片是最常用的类型，正面一个问题，背面一个答案。

**适用场景**：
- 单词记忆
- 概念定义
- 问答练习

**示例**：
```
正面：什么是间隔重复？
背面：一种基于遗忘曲线原理的学习方法，在即将遗忘时复习以巩固记忆。
```

### 变体：基本（带反向）

在类型下拉框选择"基本（带反向）"，Anki 会自动生成两张卡片：
- 正面→背面
- 背面→正面

适合需要双向记忆的场景。

## 五、插入多媒体

### 添加图片

**方法1：直接粘贴**
- 在正面或背面区域，按 `Ctrl+V` 粘贴图片

**方法2：拖拽**
- 将图片文件拖入编辑器区域

**方法3：使用菜单**
- 点击工具栏的"图片"按钮

### 添加音频

1. 点击工具栏的"音频"按钮
2. 选择音频文件
3. 音频链接会自动插入

**示例**：
```
正面：ubiquitous 发音
背面：[sound:ubiquitous.mp3]
```

复习时，点击音频图标即可播放。

### 添加 LaTeX 公式

适合数学、物理等需要公式的学科。

**行内公式**：
```
正面：$\frac{a}{b}$ 表示什么？
背面：分数，a 是分子，b 是分母
```

**块级公式**：
```
正面：$$\int_{0}^{1} x^2 dx$$ 的值是多少？
背面：$\frac{1}{3}$
```

## 六、实用技巧

### 技巧1：快捷键

| 快捷键（Windows/Linux） | 快捷键（macOS） | 功能 |
|----------------------|----------------|------|
| `Ctrl+Enter` | `Cmd+Enter` | 添加卡片 |
| `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` | 添加并继续 |
| `Ctrl+B` | `Cmd+B` | 加粗 |
| `Ctrl+I` | `Cmd+I` | 斜体 |
| `Ctrl+R` | `Cmd+R` | 切换到复习界面 |

### 技巧2：批量添加

创建多张相似卡片时：
1. 输入第一张卡片
2. 按 `Ctrl+Shift+Enter` 添加并清空
3. 继续输入下一张

### 技巧3：导入文本

如果已有大量词汇，可以用 CSV 格式批量导入：

```csv
正面,背面,标签
ubiquitous,无处不在的,英语
ephemeral,短暂的,英语
ubiquitous,发音:[juːˈbɪkwɪtəs],英语
```

## 七、常见问题

### Q: 创建的卡片在哪里？

A: 在左侧点击对应牌组，就能看到所有卡片。

### Q: 可以修改已创建的卡片吗？

A: 可以。在牌组中双击卡片，或右键选择"编辑"。

### Q: 删除的卡片能恢复吗？

A: 在浏览界面点击"恢复"，可以看到最近删除的卡片。

### Q: 卡片顺序可以调整吗？

A: 可以。拖拽卡片可以调整顺序，或在"浏览"中使用排序功能。

## 八、示例：创建10张英语单词卡片

以下是创建英语单词卡片的完整示例：

```
卡片1:
正面：ubiquitous 是什么意思？
背面：adj. 无处不在的；普遍存在的

卡片2:
正面：ephemeral 的中文是？
背面：adj. 短暂的；瞬息的

卡片3:
正面：serendipity 是什么意思？
背面：n. 意外发现美好事物的运气

卡片4:
正面：eloquent 的含义？
背面：adj. 雄辩的；口才好的

卡片5:
正面：paradigm 是什么意思？
背面：n. 范式；模式；典范

卡片6:
正面：mitigate 的中文？
背面：v. 减轻；缓和

卡片7:
正面：procrastinate 是什么意思？
背面：v. 拖延；耽搁

卡片8:
正面：ephemeral 发音
背面：[ɪˈfemərəl]

卡片9:
正面：The ___ nature of fame（填什么词？）
背面：ephemeral

卡片10:
正面：serendipity 的例句
背面：Finding that book was pure serendipity.
```

## 总结

创建 Anki 卡片非常简单：

1. 选择牌组（或新建）
2. 选择卡片类型
3. 输入正面和背面内容
4. 点击添加

关键是要保持"少而精"的原则：一张卡片，一个知识点。
