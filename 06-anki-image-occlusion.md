# Anki 图片遮挡教程：图像记忆的高效方法

图片遮挡（Image Occlusion）是 Anki 中专门用于学习图像信息的卡片类型，特别适合医学、地理、历史等需要记忆大量图像细节的学科。

## 一、什么是图片遮挡

图片遮挡的原理很简单：在图片上遮盖某些区域，让学习者回忆被遮住的内容。

**典型应用**：
- 解剖学：记忆器官位置、血管走向
- 地理：记忆省份位置、地形特征
- 历史：记忆历史事件发生地点
- 流程图：记忆业务流程步骤

## 二、前置条件

### 方法1：使用内置功能（推荐）

Anki 2.1.45+ 版本自带图片遮挡功能，无需额外安装。

### 方法2：安装插件

如果你的版本较旧，可以安装 Image Occlusion Enhanced 插件：

1. 打开 Anki → 工具 → 获取插件
2. 输入插件代码：`1374772167`
3. 重启 Anki

插件版本功能更丰富，支持：
- 批量遮挡
- 标签管理
- 更多遮挡样式

## 三、基本使用步骤

### 步骤1：创建 Cloze 类型卡片

1. 点击"添加"
2. 类型选择" Cloze "
3. 在正面区域插入图片

### 步骤2：添加图片

**方法A：粘贴**
- 复制图片到剪贴板
- 在编辑器中按 `Ctrl+V`（Windows）或 `Cmd+V`（macOS）

**方法B：拖拽**
- 将图片文件拖入编辑器区域

**方法C：菜单**
- 点击工具栏的图片按钮
- 选择图片文件

### 步骤3：框选遮挡区域

**内置版本**：
1. 点击工具栏的"图片遮挡"按钮
2. 用鼠标框选要遮挡的区域
3. 点击"隐藏"或"显示"切换遮挡样式

**插件版本**：
1. 点击工具栏的 Image Occlusion 按钮
2. 用鼠标框选多个区域
3. 选择遮挡样式（隐藏/显示）
4. 点击"添加"生成卡片

### 步骤4：保存

完成后点击"添加"保存卡片。

## 四、遮挡样式

### 样式1：隐藏（默认）

被遮挡区域显示为黑色/灰色方块，复习时需要回忆内容。

### 样式2：显示

被遮挡区域正常显示，需要说出相邻内容（适用于需要回忆相对位置的场景）。

## 五、实用场景详解

### 场景1：解剖学

**目标**：记忆人体骨骼结构

**操作**：
1. 插入全身骨骼图
2. 用图片遮挡功能框选各骨骼（如：肱骨、股骨、脊椎）
3. 每张卡片复习一个骨骼的位置

**卡片效果**：
- 卡片1：显示骨骼图，肱骨位置被遮挡
- 卡片2：显示骨骼图，股骨位置被遮挡
- ...

### 场景2：地理

**目标**：记忆中国省份位置

**操作**：
1. 插入中国地图
2. 框选各个省份区域
3. 逐个学习省份位置和名称

### 场景3：历史时间线

**目标**：记忆历史事件对应的朝代

**操作**：
1. 插入历史年表图片
2. 框选各个朝代时期
3. 学习朝代顺序和重要事件

### 场景4：流程图

**目标**：记忆业务流程步骤

**操作**：
1. 插入流程图图片
2. 框选各个步骤
3. 学习每个步骤的作用

## 六、高级技巧

### 技巧1：组合遮挡

一张图片可以框选多个区域，每个区域生成一张独立的遮挡卡片。

**示例**：一张解剖图
- 区域1：心脏位置
- 区域2：肝脏位置
- 区域3：肾脏位置

生成 3 张卡片，每张对应一个遮挡区域。

### 技巧2：命名遮挡区域

为遮挡区域添加标签，便于管理：

```
区域1：心脏
区域2：肝脏
区域3：肾脏
```

### 技巧3：批量导入

如果有大量图片需要处理，可以使用插件的批量功能。

## 七、常见问题

### Q: 图片遮挡和普通 Cloze 有什么区别？

A: 图片遮挡专注于图像记忆，适合需要记忆空间位置和视觉细节的场景。普通 Cloze 更适合文本和概念。

### Q: 可以遮挡哪些格式的图片？

A: 支持 JPG、PNG、GIF 等常见格式。

### Q: 遮挡区域可以调整大小吗？

A: 可以。在框选后，可以拖动边框调整大小。

### Q: 删除遮挡区域后能恢复吗？

A: 在保存前可以撤销。保存后需要重新创建。

### Q: 图片需要手动标注吗？

A: 是的。图片遮挡需要手动框选区域，这是为了确保学习内容的准确性。

## 八、示例：解剖学学习

**准备材料**：
- 人体解剖图（高清，标注清晰）

**创建步骤**：
1. 插入解剖图
2. 框选"心脏"区域
3. 框选"肝脏"区域
4. 框选"肾脏"区域
5. 保存生成 3 张卡片

**复习效果**：
```
卡片1：
[图片：心脏位置被遮挡]
问题：这是什么器官？

卡片2：
[图片：肝脏位置被遮挡]
问题：这是什么器官？

卡片3：
[图片：肾脏位置被遮挡]
问题：这是什么器官？
```

## 九、与 Cloze 的结合

图片遮挡可以与 Cloze 语法结合使用：

```markdown
[图片：心脏解剖图，心脏位置被遮挡]

问题：图中标注的器官是什么？
答案：心脏
```

这种组合适合需要同时记忆图像和文字的场景。
