# Anki 核心概念详解：理解间隔重复的本质

很多人在使用 Anki 时，只是机械地创建卡片、点击复习，却忽略了背后的核心原理。理解这些概念，不仅能让你用好 Anki，更能从根本上提升学习效率。

今天，我将从记忆科学的角度，深度解析 Anki 的核心概念。

## 一、遗忘曲线：为什么记忆会消失

1885 年，德国心理学家赫尔曼·艾宾浩斯通过大量实验绘制出了著名的"遗忘曲线"：

| 时间 | 记忆留存率 |
|-----|-----------|
| 20分钟后 | 58% |
| 1小时后 | 44% |
| 1天后 | 33% |
| 2天后 | 27% |
| 6天后 | 25% |
| 1个月后 | 21% |

**关键发现**：遗忘不是均匀的，而是"先快后慢"。大部分遗忘发生在学习后的 24 小时内。

这就是为什么很多人"学完就忘"——不是因为记忆力差，而是因为没有在正确的时间复习。

## 二、间隔重复：对抗遗忘的武器

既然遗忘是先快后慢，那么解决方案就是：**在即将遗忘的时候复习**。

间隔重复（Spaced Repetition）的核心思想是：
- 刚学完 → 短间隔复习
- 记得牢 → 长间隔复习
- 快忘了 → 触发复习

这样，每次复习都刚好在记忆衰减到临界点之前，有效巩固记忆。

## 三、Anki 的核心组件

理解 Anki，先理解三个基本概念：

### 1. 牌组（Deck）

牌组是卡片的容器，用于分类管理不同主题的内容。

```
📁 我的牌组
├── 📁 英语
│   ├── 托福词汇
│   └── GRE词汇
├── 📁 医学
│   ├── 解剖学
│   └── 药理学
└── 📁 编程
    ├── Python
    └── JavaScript
```

建议：牌组不宜过深，建议控制在 2-3 层。

### 2. 卡片（Card）

卡片是 Anki 的基本记忆单元，包含问题和答案。

**基础卡片结构**：
```
正面：单词 "ubiquitous"
背面：adj. 无处不在的
      例句：This technology has become ubiquitous.
```

### 3. 复习（Review）

复习是根据算法自动安排的学习过程。

**复习界面示意**：
```
Question: What is the capital of France?
-----------------------------------------
[ Show Answer ]        [ Edit ]
-----------------------------------------
Again      Hard       Good       Easy
```

点击答案后，你会看到背面，然后选择对答案的把握程度：

| 按钮 | 含义 | 效果（参考值） |
|-----|------|----------------|
| Again | 完全不记得 | 间隔重置为1分钟 |
| Hard | 有点模糊 | 间隔×1.2 |
| Good | 记起来了 | 间隔×2.5 |
| Easy | 轻松回忆 | 间隔×4.0 |

> 注：具体间隔由算法根据历史表现计算，上表为参考范围

## 四、调度算法：从 SM-2 到 FSRS

Anki 的核心是调度算法，决定每张卡片何时出现。

### SM-2：经典算法

SM-2 是 1987 年 Piotr Wozniak 开发的算法，基于一个简单的公式：

```
间隔 = 上次间隔 × 难度系数
```

优点：简单、稳定
缺点：无法适应个体差异

### FSRS：新一代算法

FSRS（Free Spaced Repetition Scheduler）由 Jarrett Ye 于 2023 年开发，基于机器学习。

FSRS 模拟三个记忆变量：
- **Retrievability（可提取性）**：某时刻能回忆起来的概率
- **Stability（稳定性）**：记忆衰减到 90% 所需的时间
- **Difficulty（难度）**：信息的固有复杂度

**实测效果**：使用 FSRS 可以减少 20-30% 的复习量，同时保持相同的记忆效果。

> 参考来源：[FSRS 官方仓库](https://github.com/open-spaced-repetition/fsrs)

## 五、主动回忆：学习的黄金法则

Anki 之所以有效，核心在于**主动回忆**。

### 被动学习 vs 主动回忆

| 方式 | 示例 | 效果 |
|-----|------|------|
| 被动学习 | 反复阅读笔记 | ★☆☆☆☆ |
| 被动识别 | 看单词书认意思 | ★★☆☆☆ |
| 主动回忆 | 闭眼回忆答案 | ★★★★★ |

研究表明，主动回忆的记忆效果是被动学习的 **2-3 倍**。

Anki 的设计正是基于这一原理：每次复习都是一次主动回忆练习。

## 六、优质卡片制作

## 七、常见误区

理解了核心概念，我们来看看常见的误区：

### 误区1：卡片越多越好

100 张精心制作的卡片，远胜 1000 张粗糙的卡片。

### 误区2：只看记得住的内容

如果一张卡片总是轻松回答，说明它已经记住了，可以考虑暂停或删除。

### 误区3：忽视"再次"按钮

遇到不会的卡片时，不要轻易跳过。Again 按钮是在帮你巩固记忆。

### 误区4：追求完美的时间点

复习的核心是"在遗忘前"，而不是"精确的某个时间点"。偶尔晚几分钟复习不会影响效果。

## 总结

Anki 的核心概念可以归结为：

1. **遗忘曲线**：记忆会先快后慢地衰减
2. **间隔重复**：在即将遗忘时复习
3. **牌组与卡片**：组织和管理知识的基本单元
4. **调度算法**：SM-2 或 FSRS 自动计算最优复习时间
5. **主动回忆**：比被动学习更有效的记忆方式

理解这些概念，你不仅会用 Anki，更会用好 Anki。
