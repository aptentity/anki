# Anki 入门完全指南：从小白到达人

Anki 是一款强大的间隔重复记忆软件，被全球医学生、语言学习者和考试备考者广泛使用。但对于新手来说，如何开始使用 Anki 往往是个难题。

本文将从零开始，手把手教你掌握 Anki 的基本使用方法。

## 一、什么是 Anki

Anki 是一款基于间隔重复原理的记忆工具。它的核心理念是：**在即将遗忘的时候复习，以达到最佳的记忆效果**。

### 为什么选择 Anki？

| 特点 | 说明 |
|-----|------|
| 间隔重复 | 基于遗忘曲线，科学安排复习时间 |
| 开源免费 | 桌面端完全免费，代码开放 |
| 跨平台 | 支持 Windows、Mac、Linux、Android、iOS |
| 自定义强 | 支持各种卡片类型、插件、模板 |
| 云同步 | 支持 AnkiWeb 云同步，多设备无缝切换 |

### 适用场景

- **语言学习**：背单词、学语法、练口语
- **考试备考**：医学、法律、工程师资格证
- **职业培训**：编程、法律条文、财务知识
- **兴趣爱好**：历史、地理、音乐、摄影

## 二、界面介绍

### 主界面概览

打开 Anki，你会看到主界面：

```
┌─────────────────────────────────────────┐
│  Anki                                     │
├─────────────────────────────────────────┤
│  [添加] [浏览] [统计] [导出]              │
├─────────────────────────────────────────┤
│                                          │
│  📚 学习                                │
│  ┌─────────────────────────────────────┐ │
│  │ 英语词汇         0 / 15             │ │

│  └─────────────────────────────────────┘ │
│                                          │

│  今日新卡片: 20    今日复习: 45          │
│                                          │
└─────────────────────────────────────────┘
```

### 核心概念

| 概念 | 说明 |
|-----|------|
| **牌组（Deck）** | 存放卡片的容器，类似文件夹 |
| **卡片（Card）** | 正面+背面的记忆单元 |
| **笔记（Note）** | 创建卡片的内容源 |
| **复习（Review）** | 根据算法出现的练习 |
| **学习（Larning）** | 新卡片的学习阶段 |
| **重新学习（Relearning）** | 忘记后的再学习阶段 |

## 三、创建第一张卡片

### 方法1：手动添加

1. 点击顶部的 **"添加"** 按钮
2. 在正面输入问题，如：`Hello`
3. 在背面输入答案，如：`你好`
4. 选择牌组（默认"默认"）
5. 点击 **"添加"**

### 方法2：快捷键添加

- 按 `A` 键快速打开添加界面
- 按 `Shift+A` 快速打开当前牌组的添加界面

### 卡片类型

Anki 支持多种卡片类型：

**1. 基础卡片**
```
正面：Hello
背面：你好
```

**2. 基础（带反向）**
自动生成两张卡片：
- 卡片1：Hello → 你好
- 卡片2：你好 → Hello

**3. Cloze 完形填空**
```
{{c1::Hello}} 的中文意思是 {{c2::你好}}
```
显示为：`[___] 的中文意思是 [___]`

## 四、复习流程

### 开始复习

1. 在主界面点击一个牌组
2. 点击 **"学习"** 按钮
3. 进入复习界面

### 复习界面

```
┌─────────────────────────────────────────┐
│  Hello                                  │
│                                         │
│  ─────────────────────────────────────  │
│                                         │
│  [显示答案]                              │
└─────────────────────────────────────────┘
```

点击"显示答案"后：

```
┌─────────────────────────────────────────┐
│  Hello                                  │
│                                         │
│  ─────────────────────────────────────  │
│                                         │
│  你好                                    │
│                                         │
│  [重来] [困难] [良好] [简单]              │
└─────────────────────────────────────────┘
```

### 按钮含义

| 按钮 | 含义 | 效果 |
|-----|------|------|
| **重来** | 完全忘记 | 卡片重新进入学习阶段 |
| **困难** | 记得但很吃力 | 稍短间隔 |
| **良好** | 正常记住 | 正常间隔 |
| **简单** | 太简单了 | 稍长间隔 |

### 间隔预览

当你点击按钮时，Anki 会显示下次复习的时间：

| 按钮 | 首次间隔（默认） |
|-----|----------------|
| 重来 | 1 分钟 |
| 困难 | 10 分钟 |
| 良好 | 1 天 |
| 简单 | 4 天 |

## 五、牌组管理

### 创建牌组

1. 点击顶部的 **"添加"** 旁的下拉菜单
2. 选择 **"创建牌组"**
3. 输入牌组名称，如：`英语`
4. 点击 **"确定"**

### 子牌组

使用 `::` 创建层级结构：

| 牌组名称 | 层级 |
|---------|------|
| `英语` | 一级 |
| `英语::托福` | 二级（托福是英语的子牌组）|
| `英语::托福::阅读` | 三级 |

### 牌组选项

右键点击牌组 → 选择"选项"可以设置：

- 每日新卡片数量
- 每日复习上限
- 学习步数
- 切换算法（SM-2 / FSRS）

## 六、搜索与筛选

### 基础搜索

在浏览器中，可以使用以下搜索语法：

| 语法 | 说明 | 示例 |
|-----|------|------|
| `deck:牌组名` | 指定牌组 | `deck:英语` |
| `tag:标签名` | 指定标签 | `tag:动词` |
| `is:new` | 新卡片 | `is:new` |
| `is:learn` | 学习中 | `is:learn` |
| `is:review` | 待复习 | `is:review` |

### 组合搜索

```
deck:英语 tag:动词 is:new
```
查找英语牌组中标签为"动词"的新卡片。

### 全文搜索

在搜索框输入关键词，会搜索卡片正反面所有内容。

## 七、标签系统

### 添加标签

在添加/编辑卡片时，底部有标签输入框：

```
标签: 动词, 核心词汇, 高频
```

### 使用标签

- **筛选**：在浏览器中按 `tag:标签名` 筛选
- **批量操作**：选中多张卡片后批量添加/删除标签
- **智能牌组**：使用搜索条件创建"智能牌组"自动更新

### 标签命名建议

- 使用英文或拼音（避免编码问题）
- 层级用 `::` 分隔，如 `英语::词汇::动词`

## 八、同步与备份

### AnkiWeb 同步

1. 注册 AnkiWeb 账号：https://ankiweb.net
2. 在 Anki 中：工具 → 同步
3. 登录账号，点击"同步"
4. 之后每次打开/关闭 Anki 会自动同步

### 本地备份

1. 工具 → 创建备份
2. 自动保存最近的备份
3. 备份文件位置：文档/Anki2/用户/backups/

### 导出与导入

- **导出**：文件 → 导出 → 选择牌组 → 保存为 .apkg 文件
- **导入**：双击 .apkg 文件，或文件 → 导入

## 九、实用技巧

### 1. 快捷键

| 快捷键 | 功能 |
|-------|------|
| `A` | 添加卡片 |
| `B` | 浏览 |

**添加图片**：
- 直接粘贴（Ctrl+V）

- 工具栏的音频按钮录制
- 或直接粘贴音频文件

## 十、常见问题

### Q: 复习积压太多怎么办？

A: 在牌组选项中降低每日新卡片数量，暂时跳过部分复习（非永久删除，只是暂时隐藏）。

### Q: 忘记的卡片太多怎么办？

A: 使用"忘记卡片"筛选，将太久的卡片重置为新卡片重新学习。

### Q: 如何制作高质量卡片？

A: 遵循以下原则：
- 一次只问一个问题
- 保持卡片简洁
- 使用自己的话，不要复制粘贴
- 添加图片和例句增强理解

### Q: Anki 太占用时间怎么办？

A: 设置每日限制：
- 新卡片：10-20 张
- 复习上限：100-200 张
- 完成后可以继续，但建议控制总量

### Q: 数据丢失怎么办？

A: 
- 定期同步到 AnkiWeb
- 定期创建本地备份
- 导出重要牌组为 .apkg 文件

Anki 的学习曲线并不陡峭，关键在于：

1. **坚持复习**：每天都要复习，不要中断
