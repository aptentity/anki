浏览窗口让你搜索卡片和笔记并进行编辑。点击主窗口中的 **Browse** 或按 `B` 即可打开。它由三部分组成：

- 左侧**侧边栏**
- 右上**卡片 / 笔记表格**
- 右下**编辑区**

把鼠标放在两个区域之间，可以拖动调整大小。

## 表格模式

Anki 2.1.45+ 提供两种模式：数据表格中显示**卡片**或**笔记**。你可以通过点击搜索区域左侧的切换按钮，或按 `Ctrl`+`Alt`+`T`（Mac 上为 `Cmd`+`Opt`+`T`）来切换。切换按钮也指示当前显示的是 **C**ards 还是 **N**otes。

> 📌 为简单起见，本手册默认假设 Cards 模式处于活动状态。当提到"选择 / 查找卡片"时，你可以根据当前活动模式替换为"卡片或笔记"。

## 侧边栏

左侧的**侧边栏**提供常用搜索项的快捷访问。在 Anki 2.1.45+ 上，它还提供搜索栏、编辑标签和牌组的工具，以及两种不同的工具（见下文）。你可以用侧边栏顶部的工具栏切换工具，或用快捷键 `Alt`+`1`/`2`。

### 搜索工具

使用此工具时，侧边栏的行为与旧版本相同：点击项目会执行搜索。

- 按住 `Ctrl`（Mac 上为 `Command`）点击项目，可以将点击的项目以 **AND** 条件追加到当前搜索，而不是开始新搜索。例如：点击"Learning"，然后 `Ctrl`-点击"German"，可以显示同时属于两者的卡片。
- 按住 `Shift` 可以创建 **OR** 搜索。例如：点击一个牌组，然后 `Shift`-点击另一个牌组，会显示来自任一牌组的卡片。
- 按住 `Alt`（Mac 上为 `Option`）可以**反转**搜索（前面加 `-`）：例如，显示当前牌组中**没有**某个标签的所有卡片。`Alt`/`Option` 可以和 `Ctrl` 或 `Shift` 组合使用。

在 Anki 2.1.39+ 上，你还可以同时按住 `Ctrl` 和 `Shift` 点击搜索项，把同一类型的搜索全部替换为新的。例如：你之前输入了复杂的搜索表达式 `deck:Swahili (is:due or tag:important)`，现在想对乌尔都语牌组执行同样的搜索。可以按住 `Ctrl`+`Shift` 点击侧边栏中的 Urdu 牌组，得到：`deck:Urdu (is:due or tag:important)`。

### 选择工具

选择工具允许通过按住 `Ctrl` 或 `Shift` 点击来**同时选择多个项目**。它还支持拖放来重新排序牌组和标签。

示例：假设你有标签 `Math`、`Calculus` 和 `Algebra`。点击 `Calculus` 标签，然后 `Ctrl`-点击 `Algebra` 标签。现在两个标签都被选中了，把其中一个拖到 `Math` 标签上，使它们都成为 `Math` 的子标签。Anki 在后台会将两个标签重命名为 `Math::Calculus` 和 `Math::Algebra`，并相应更新你的笔记。

另一个选择多个项目的用途是搜索：在选中的项目上右键，可以选择 **Search → All/Any Selected**。这也可以与搜索工具中描述的键盘修饰符组合，将结果追加到当前搜索。

### 已保存的搜索

如果你经常搜索同样的内容，可以右键点击侧边栏最顶端的项目，选择 **Save Current Search** 并输入名称来保存当前搜索。你也可以把任何侧边栏项目拖放到这个区域，添加一个等效的已保存搜索，相当于把它固定在顶部。

### 编辑项目

你可以直接从侧边栏（或右键菜单，或用快捷键 `Del` 和 `F2`）删除或重命名标签、牌组和已保存的搜索。删除也支持同时操作多个项目（参见[选择工具](#选择工具)）。

### 查找项目

要在侧边栏树中找到某个项目，在顶部的搜索栏中输入部分名称即可。

## 搜索框

卡片列表上方有一个搜索框。你可以在里面输入各种内容来搜索卡片。搜索语法详见[搜索](https://docs.ankiweb.net/searching.html)。

## 卡片 / 笔记表格

表格的每一行代表匹配当前搜索的一张卡片或一条笔记。点击某一行时，对应的笔记会在下方区域显示。

### 行

如果拖动鼠标或按住 `Ctrl` 或 `Command` 选择多行，编辑器会暂时隐藏。各种操作（如更改牌组）可以**同时**对多张卡片或笔记生效，与活动模式无关。因此：

- 在 Cards 模式下，如果任何卡片被选中，其笔记就被视为被选中
- 在 Notes 模式下，如果笔记被选中，其卡片就被视为被选中

其他操作（如显示卡片信息）只对单张卡片或单条笔记生效。这称为当前卡片或当前笔记，通常是最后选中或点击的那一个。

行背景颜色会根据卡片和笔记状态变化。在 Cards 模式下，使用第一个匹配的：

1. 如果卡片被**标记**，使用标记颜色
2. 如果卡片被**暂停**，黄色
3. 如果卡片的笔记被**标记**，紫色

在 Notes 模式下，颜色只应用于被标记的笔记。

### 列

列是可配置的：在某列上右键（Mac 上 `Ctrl`-点击）可以选择你想显示哪些列。拖动列可以重新排序。点击某列会按该列排序；再次点击反向排序。

> ⚠️ 你无法按 Question 和 Answer 列排序。

两种模式下的列行为：

| 列 | Cards 模式 | Notes 模式 |
|---|---|---|
| **Answer** | 卡片的背面，一行，去掉问题。你也可以在卡片类型编辑器中设置[自定义格式](https://docs.ankiweb.net/templates/styling.html#browser-appearance)。 | 与 Cards 模式相同，仅针对笔记的第一张卡片。 |
| **Card(s)** | 卡片模板的名称。 | 该笔记的卡片数量。 |
| **Card Modified** | 上一次对卡片做更改的时间（如复习后更新了复习历史和间隔）。 | 该笔记某张卡片最近一次更改的时间。 |
| **Created** | 与 Cards 模式相同。 | 笔记创建日期。 |
| **Deck** | 卡片所在牌组的名称。 | 笔记卡片所在的不同牌组数量，或全部在同一牌组时显示牌组名。 |
| **Due** | 复习/（重新）学习卡片的到期日，新卡片显示在新卡片队列中的位置。被暂停或埋葬的卡片用括号括起。排序先按类型，再按日期或位置。 | 该笔记下一张未暂停/未埋葬/不在筛选牌组中的到期复习或（重新）学习卡片。 |
| **(Avg.) Ease** | 非新卡片的容易度。 | 该笔记非新卡片的平均容易度。 |
| **(Avg.) Interval** | 在复习或重新学习中的卡片的间隔。 | 该笔记在复习或重新学习中的卡片的平均间隔。 |
| **Lapses** | 卡片被评"Again"的次数。 | 该笔记所有卡片的总失误次数。 |
| **Note** | 与 Cards 模式相同。 | 笔记类型的名称。 |
| **Note Modified** | 与 Cards 模式相同。 | 上一次编辑笔记内容的时间。 |
| **Reposition** | 更改新卡片出现的顺序。你可以通过启用 *due* 列来查看现有位置。如果对多张卡片执行重新定位命令，会依次给每张卡片递增编号。**Shift position of existing cards** 选项允许在现有卡片之间插入新卡片，把现有卡片推开。例如：你有 5 张卡片，想把 3、4、5 移到 1 和 2 之间，选中此设置后，卡片最终顺序为 1、3、4、5、2。关闭此选项时，1 和 2 会得到相同的编号。⚠️ 启用后，任何位置号更高的卡片都会被修改，同步时需要把这些更改一起发送。 |
| **Toggle Suspend** | [暂停](https://docs.ankiweb.net/studying.html#editing-and-more)或取消暂停所有选中的卡片，取决于当前卡片是否已暂停。 | 同 Cards 模式。 |
| **Flag** | 切换所有选中卡片的标记。是否添加或移除标记取决于当前卡片是否有选中的标记。 | 同 Cards 模式。 |
| **Info** | 显示当前卡片的各种信息，包括复习历史。详见[卡片信息](https://docs.ankiweb.net/stats.html#card-info)。 | 同 Cards 模式。 |

## 编辑区

当在表格中选择一行时，对应笔记的内容会显示在底部编辑区。你可以在此直接修改字段内容。

## 菜单和操作

### 编辑

- **Select All**：`Ctrl`+`A` — 全选
- **Select None**：`Ctrl`+`Shift`+`A` — 取消选择
- **Invert Selection** — 反选
- **Undo**：`Ctrl`+`Z` — 撤销
- **Redo**：`Ctrl`+`Shift`+`Z` — 重做

### 笔记

- **Add Notes**：`Ctrl`+`N` — 添加笔记
- **Duplicate**：`Ctrl`+`D` — 复制笔记
- **Delete**：`Del` — 删除
- **Change Note Type** — 更改笔记类型
- **Find Duplicates** — 查找重复

### 卡片

- **Add Cards** — 添加卡片
- **Change Deck** — 更改牌组
- **Change Card Type** — 更改卡片类型
- **Find and Replace** — 查找和替换
- **Reposition** — 重新定位
- **Toggle Suspend** — 暂停/取消暂停
- **Flag** — 标记

### 跳转

此菜单提供跳转到浏览器各部分的键盘快捷键，以及在卡片列表中上下移动。

## 查找和替换

此对话框允许替换笔记中的文本。如上所述，可从工具栏和表格的右键菜单中打开。

第一个输入框是要替换的文本，第二个是替换文本。下拉菜单允许你指定 Anki 应该在哪里查找文本：笔记的标签（需要 Anki 2.1.45+）、所有字段，或仅在特定字段中（只有被选中笔记的字段才会列出）。

默认情况下，只影响选中的笔记。如果想取消此限制，可以取消勾选"selected notes only"（需要 Anki 2.1.45+）。

**正则表达式**选项允许你执行复杂的替换。例如，假设字段中有以下文本：

```html
<img src="pic.jpg" />
```

使用以下设置：

![查找和替换对话框](media/find_and_replace.png)

（注：在 Anki 2.1.28 之前的版本中，需要把 `${1}` 替换为 `\1`。）

字段内容会变为：

```
pic.jpg
```

正则表达式的完整讨论超出了本手册的范围。网上有大量语法指南：

- 旧版 Anki：[Python re 模块](http://docs.python.org/library/re.html)

## 查找重复

你可以用 **Notes → Find Duplicates** 选项搜索内容相同的笔记。打开窗口后，Anki 会查看你所有的笔记类型并列出所有可能的字段。如果你想在 *Back* 字段中查找重复，选择该字段然后点击 **Search**。

默认情况下，它会在所有包含该字段的笔记类型中搜索。这与手动添加卡片时的重复检查不同——后者只限于单个笔记类型。

**Optional filter** 文本框允许你缩小搜索范围。例如，只想搜索"French Vocab"和"French Verbs"笔记类型中的重复，输入：

```
"note:french vocab" or "note:french verbs"
```

或者只想在特定牌组中搜索：

```
"deck:myDeck"
```

搜索语法与浏览器中的搜索相同。更多信息请参见[搜索](https://docs.ankiweb.net/searching.html)。

你可以在搜索结果列表中点击某个链接，在该集合中显示重复笔记。如果结果数量很大，可以点击 **Tag Duplicates** 按钮，用 *duplicate* 标签标记所有匹配的笔记。然后在浏览器中搜索这个标签，可以统一处理。
