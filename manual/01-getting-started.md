## 核心概念

### 卡片

问题和答案配对称为*卡片*。它类似于纸质抽认卡，正面是问题，背面是答案。但是，在 Anki 中，卡片看起来不像实体卡片，当你显示答案时，默认情况下问题仍然可见。例如，如果你正在学习基础化学，你可能会看到这样的问题：

```
Q: 氧气的化学符号是什么？
```

在确定答案是 O 后，点击 **显示答案** 按钮，Anki 会显示：

```
Q: 氧气的化学符号是什么？
A: O
```

确认答案正确后，你告诉 Anki 你记得多好，Anki 会选择何时再次向你展示这张卡片。例如，Anki 可能决定 3 天后再向你展示这张卡片。在这种情况下，我们称这张卡片的间隔为 3 天。

#### 卡片状态

- **新卡片：** 你下载或自己创建的，但从未学习过的卡片。
- **学习中：** 最近才看到且仍在学习的卡片。
- **复习中：** 已经完成学习的卡片。这些卡片会在间隔时间到期后再次显示。复习卡片有两种类型：
    - **幼稚卡片：** 间隔小于 21 天的卡片。
    - **成熟卡片：** 间隔为 21 天或更长的卡片。
- **重新学习：** 在复习阶段遗忘的卡片。这些卡片会返回到重新学习状态重新学习。

### 牌组

*牌组*是一组卡片。你可以将卡片放在不同的牌组中，这样就可以只学习卡片集合的一部分，而不是一次学习所有内容。每个牌组可以有不同的设置，例如每天显示多少张新卡片，或者等待多长时间才再次显示卡片。

牌组可以包含其他牌组，这样你就可以将牌组组织成树状结构。Anki 使用双冒号（"::"）来表示牌组树中的不同层级。例如，名为"Chinese::Hanzi"的牌组指的是一个"Hanzi"牌组，它是"Chinese"牌组的一部分。如果你选择"Hanzi"，则只会显示 Hanzi 卡片；如果你选择"Chinese"，则所有中文卡片都会显示，包括 Hanzi 卡片。

要将牌组放入树中，你可以用双冒号分隔每个层级来命名它们，也可以在牌组列表中拖放它们。放在另一个牌组内的牌组通常称为"子牌组"，顶层牌组称为"父牌组"。

Anki 默认附带一个名为"默认"的牌组；任何与其他牌组分离的卡片都会进入这里。如果默认牌组不包含任何卡片且你已添加其他牌组，Anki 会隐藏默认牌组。或者，你可以重命名这个牌组并将其用于其他卡片。

牌组列表中的牌组按字母顺序排序。如果你的牌组名称包含数字，可能会导致令人惊讶的顺序。例如，"My Deck 10" 会排在 "My Deck 9" 之前，因为 1 在 9 之前。如果你想让 "My Deck 9" 排在前面，可以将其重命名为 "My Deck 09"，这样它就会出现在 "My Deck 10" 之前。

牌组最适合用于存放卡片的广泛类别，而不是"食物动词"或"第一课"这样的具体主题。有关详细信息，请参阅[合理使用牌组](https://docs.ankiweb.net/editing.html#using-decks-appropriately)部分。

有关牌组顺序如何影响卡片学习顺序的信息，请参阅[显示顺序](https://docs.ankiweb.net/studying.html#display-order)部分。

### 笔记与字段

制作抽认卡时，通常希望制作多张与同一信息相关的卡片。例如，如果你正在学习法语，你学到单词 *bonjour* 的意思是 hello，你可能希望创建一张卡片显示"bonjour"让你记住"hello"，另一张卡片显示"hello"让你记住"bonjour"。一张卡片测试你识别法语单词的能力，另一张卡片测试你产生它的能力。

使用纸质抽认卡时，在这种情况下你唯一的选择是将信息写两遍，每张卡片一次。一些抽认卡程序提供了翻转正面和背面的功能，使这变得更方便。但这相比纸质卡片改进了有限，而且有两个主要缺点：

- 因为这些程序不会分别追踪你的识别和产生表现，卡片往往不会在最佳时间显示给你，这意味着你会比预期忘记更多，或者学习超过必要的量。
- 反转问题和答案只有在每面想要完全相同的内容时才有效。这意味着例如不可能在每张卡片的背面显示额外信息。

Anki 通过允许你将卡片内容分割成独立的信息片段来解决这些问题。然后你可以告诉 Anki 你希望在每张卡片上显示哪些信息片段，Anki 会负责为你创建卡片，并在你将来进行任何编辑时更新它们。

想象一下我们想学习法语词汇，并且我们想在每张卡片的背面包含一个教科书页码。我们希望卡片看起来像这样：

```
Q: Bonjour
A: Hello
 页码 #12
```

和这样：

```
Q: Hello
A: Bonjour
 页码 #12
```

在这两张卡片中，我们都有三个相关的信息片段：法语单词、英语释义和页码。如果我们将它们放在一起，它们会是这样：

```
法语: Bonjour
英语: Hello
页码: 12
```

在 Anki 中，这种相关信息集合称为*笔记*，每个信息片段包含在*字段*中。在这个例子中，笔记有三个字段："法语"、"英语"和"页码"。

要添加和编辑字段，请在添加或编辑笔记时点击**字段...**按钮。有关字段的更多信息，请参阅[自定义字段](https://docs.ankiweb.net/editing.html#customizing-fields)部分。

### 卡片类型

为了让 Anki 根据我们的笔记创建卡片，我们需要给它一个蓝图，说明每个卡片的正面或背面应显示哪些字段。这个蓝图称为*卡片类型*。每种类型的笔记可以有一个或多个卡片类型；当你添加笔记时，Anki 会为每个卡片类型创建一张卡片。

所有卡片类型都有两个*模板*，一个用于问题，一个用于答案。在之前的法语示例中，我们希望识别卡的背面看起来像这样：

```
Q: Bonjour
A: Hello
 页码 #12
```

为此，我们可以将答案模板设置为：

```
Q: {{法语}}
A: {{英语}}

 页码 #{{页码}}
```

在卡片模板中，字段名称用双花括号包裹，如 `{{法语}}` 或 `{{英语}}`。Anki 会用字段中包含的实际文本替换它们。这称为[字段替换](https://docs.ankiweb.net/templates/fields.html)。未用双花括号包裹的文本在每张卡片上都是相同的。例如，我们不需要在每个笔记中添加"页码 #"，因为模板会自动将其添加到每张卡片上。`<br>` 标签是一个特殊代码，告诉 Anki 移动到下一行。详细信息请参阅[模板](https://docs.ankiweb.net/templates/intro.html)部分。

产生卡的模板也将以类似的方式工作：

```
Q: {{英语}}
A: {{法语}}

 页码 #{{页码}}
```

创建卡片类型后，每次你添加新笔记时，都会基于该卡片类型创建一张卡片。卡片类型可以轻松保持卡片格式的一致性，并大大减少添加信息所需的工作量。它们还意味着 Anki 可以确保相关卡片不会太接近地出现，并且它们允许你修复一次打字错误或事实错误，然后所有相关卡片会立即更新。

要添加和编辑卡片类型，请在添加或编辑笔记时点击**卡片...**按钮。有关卡片类型的更多信息，请参阅[卡片和模板](https://docs.ankiweb.net/templates/intro.html)部分。

### 笔记类型

Anki 允许你为不同类型的材料创建不同类型的笔记。每种类型的笔记都有自己的字段集和卡片类型。为你学习的每个广泛主题创建一个单独的笔记类型是个好主意。在之前的法语示例中，我们可能会为此创建一个名为"法语"的笔记类型。如果我们想学习首都城市，我们也可以为此创建一个笔记类型，包含"国家"和"首都城市"等字段。

Anki 附带了一些标准笔记类型。这些笔记类型是为了让新用户更容易使用 Anki，但长期来看，建议你为自己学习的内容创建专门的笔记类型。标准笔记类型有：

- **Basic（基础）**
  有"正面"和"背面"字段，将创建一张卡片。在"正面"中输入的文本将显示在卡片正面，在"背面"中输入的文本将显示在卡片背面。

- **Basic (and reversed card)（基础及反向卡片）**
  与"基础"类似，但会为你输入的文本创建两张卡片：正面→背面和背面→正面。

- **Basic (optional reversed card)（基础可选反向卡片）**
  与"基础"类似，但有一个名为"添加反向"的第三个字段。如果你在该字段中输入任何文本，也会创建一张反向卡片（背面→正面）。详细信息请参阅[卡片和模板](https://docs.ankiweb.net/templates/intro.html)部分。

- **Basic (type in the answer)（基础输入答案）**
  本质上是"基础"，但正面有一个额外的文本框，你可以输入答案。显示背面时，Anki 会显示你的输入与实际答案之间的差异。详细信息请参阅[检查你的答案](https://docs.ankiweb.net/templates/fields.html#checking-your-answer)部分。

- **Cloze（填空）**
  一种笔记类型，允许你选择文本并将其转换为填空删除（例如，"人类在[……]登陆月球"→"人类在1969年登陆月球"）。详细信息请参阅[填空删除](https://docs.ankiweb.net/editing.html#cloze-deletion)部分。

- **Image Occlusion（图片遮挡）**
  与填空笔记类型类似，但使用图片而不是文本，这在学习严重依赖图片的材料（如解剖学和地理）时特别有用。详细信息请参阅手册的[图片遮挡](https://docs.ankiweb.net/editing.html#image-occlusion)部分。

要添加你自己的笔记类型或修改现有的笔记类型，你可以从 Anki 主窗口使用**工具 > 管理笔记类型**。

笔记和笔记类型是整个集合共有的，而不是仅限于单个牌组。这意味着你可以在单个牌组中使用不同的笔记类型，或者将从同一笔记生成的卡片放到不同的牌组中。当你使用添加窗口添加笔记时，你可以选择使用什么笔记类型和放入什么牌组，这些选择是完全独立的。你也可以在创建笔记后[更改笔记的笔记类型](https://docs.ankiweb.net/browsing.html#notes)。

### 集合

你的*集合*是存储在 Anki 中的所有材料：你的卡片、笔记、牌组、笔记类型、牌组设置等。

## 共享牌组

你可以在 YouTube 上观看关于[共享牌组与复习基础](http://www.youtube.com/watch?v=QS2G-k2hQyg&yt:cc=on)的视频。

开始使用 Anki 最简单的方法是下载别人分享的卡片牌组：

1. 点击牌组列表底部的**获取共享**按钮。

2. 当你找到感兴趣的牌组时，点击**下载**按钮下载牌组包。

3. 双击下载的包以导入到 Anki，或转到**文件 > 导入**。

注意：目前无法直接将共享牌组添加到你的 AnkiWeb 账户。你需要先将其导入桌面应用 Anki、AnkiMobile 或 AnkiDroid，然后[同步](https://docs.ankiweb.net/syncing.html)将牌组上传到 AnkiWeb。

创建你自己的牌组是学习复杂主题最有效的方式。语言和科学等学科不能仅靠记忆事实来理解——你需要解释和背景才能有效地学习它们。此外，自己输入信息会迫使你决定关键点是什么，从而带来更好的理解。

如果你是一名语言学习者，你可能会想下载一长串单词及其翻译，但这不会教你语言，就像记忆科学方程式不会教你天体物理学一样。要正确学习，你可能需要教科书、老师或接触真实世界的句子。

```
如果你不理解，就不要学习。
--SuperMemo
```

大多数共享牌组是由人们在 Anki 之外学习材料（如教科书、课程、电视等）时创建的。他们从所学内容中选择有趣的点放入 Anki。他们可能不会在卡片上添加任何背景信息或解释，因为他们已经理解了材料。所以当其他人下载他们的牌组并尝试使用时，可能会发现由于缺少背景信息和解释而非常困难。

这并不是说共享牌组没用。如果你在学习教科书 ABC，而有人分享了 ABC 想法的牌组，这是节省时间的好方法。对于基本上是事实列表的简单主题（如首都城市名称或国旗），你可能不需要任何外部材料。然而，对于复杂的主题，共享牌组应该作为外部材料的*补充*，而不是*替代*。
