## 添加卡片和笔记

回忆一下基础知识：在 Anki 中我们添加的是**笔记**（note），而不是卡片，卡片是由 Anki 为我们自动生成的。点击主窗口中的"添加"按钮，就会出现"添加笔记"窗口。

![添加笔记窗口](https://docs.ankiweb.net/media/add_screen.png)

窗口左上角显示的是当前的笔记类型（note type）。如果没有显示"基本"，那可能是因为你在下载共享牌组时添加了一些笔记类型。下面的说明假设选中的是"基本"类型。

窗口右上角显示的是卡片将要添加到的牌组。如果你想把卡片添加到一个新牌组中，可以点击牌组名称按钮，然后点击"添加"。

在笔记类型下方，你会看到一些按钮和一个标记为"正面"和"背面"的区域。正面和背面称为**字段**（fields），你可以通过点击上方的"字段…"按钮来添加、删除和重命名它们。

在字段下方是另一个标记为"标签"的区域。**标签**（tags）是你可以附加到笔记上的标签，以便更轻松地组织和查找笔记。如果需要的话，你可以将标签留空，也可以添加一个或多个。标签之间用空格分隔。如果标签区域显示：

```
vocab check_with_tutor
```

……那么你添加的笔记将有两个标签。

在正面和背面输入文本后，你可以点击"添加"按钮或按 `Ctrl+Enter`（Mac 上为 `Command+Enter`）将笔记添加到你的卡片集中。这样做时，会同时创建一张卡片，并放入你选择的牌组中。如果你想编辑已添加的卡片，可以点击历史按钮在浏览器中搜索最近添加的卡片。

有关笔记类型和字段之间按钮的更多信息，请参阅编辑器部分。

### 重复检查

Anki 会检查第一个字段是否唯一，因此如果你输入两张正面字段为"apple"的卡片（例如），它会发出警告。唯一性检查仅限于当前笔记类型，因此如果你同时学习多种语言，只要每种语言使用不同的笔记类型，两张正面相同的卡片就不会被列为重复。

为了效率考虑，Anki 不会自动检查其他字段中的重复，但浏览器中有一个"查找重复"功能，你可以定期运行。

### 有效学习

不同的人喜欢不同的复习方式，但有一些通用的概念需要牢记。一个很好的入门资料是 SuperMemo 网站上的这篇文章。特别要注意：

**保持简洁**：卡片越短，复习起来越容易。你可能很想"以防万一"地塞入大量信息，但复习很快就会变得痛苦。

**不要在没有理解的情况下死记硬背**：如果你正在学习一门语言，尽量避免大段地罗列单词。学习语言最好的方法是在上下文中学习，也就是说要在句子中看到这些单词的用法。同样，假设你正在学习一门计算机课程。如果你试图死记硬背一大堆缩写词，你会发现很难取得进步。但是，如果你花时间理解缩写词背后的概念，记住这些缩写词就会变得容易得多。

## 添加笔记类型

虽然基本的笔记类型对于只有单词或短语的简单卡片来说已经足够了，但一旦你发现需要在正面或背面包含多于一条信息时，最好将这些信息拆分到更多字段中。

你可能会想："我只想要一张卡片，为什么不能把音频、图片、提示和翻译都放在正面字段里呢？"如果你更喜欢这样做，那当然可以。但这种方法的缺点是所有信息都粘在一起了。如果你想按提示对卡片进行排序，你就无法做到这一点，因为它与其他内容混杂在一起。你也无法执行诸如将音频从正面移动到背面之类的操作，除非为每条笔记都费力地复制和粘贴。通过将内容放在不同的字段中，你可以更轻松地在未来调整卡片的布局。

要创建新类型的笔记，请在主 Anki 窗口中选择"工具 > 管理笔记类型"。然后点击"添加"以添加新类型的笔记。你将看到另一个屏幕，可以选择要基于哪种笔记类型来创建新类型。"添加"表示基于 Anki 自带的类型来创建新类型。"克隆"表示基于你卡片集中已有的类型来创建新类型。例如，如果你已经创建了一个法语词汇类型，你可能希望在创建德语词汇类型时克隆它。

选择"确定"后，系统会要求你为新类型命名。命名时选择你正在学习的主题材料是一个不错的选择——比如"日语"、"常识"等等。选择好名称后，关闭笔记类型窗口，你将返回到添加窗口。

## 自定义字段

要在添加或编辑笔记时自定义字段，请点击"字段…"按钮；或者在管理笔记类型窗口中选中笔记类型时也可以点击。

![字段对话框](https://docs.ankiweb.net/media/fields.png)

你可以通过点击相应的按钮来添加、删除或重命名字段。

要更改字段在此对话框和添加笔记对话框中出现的顺序，你可以使用"重新定位"按钮，它会要求你输入希望字段所在的数字位置。因此，如果你想将某个字段更改为新的第一个字段，请输入"1"。

或者你也可以拖放字段名称来重新排序。要执行此操作，请使用鼠标或手指将字段拖动到所需位置。指示器将显示字段将被移动到的位置。

**不要**将"Tags"、"Type"、"Deck"、"Card"或"FrontSide"用作字段名，因为它们是特殊字段，无法正常工作。

屏幕底部的选项允许你编辑添加和编辑卡片时要使用的字段的各种属性。这不是自定义复习时卡片上显示内容的地方；如需自定义，请参阅模板。

**编辑字体**允许你自定义编辑笔记时使用的字体和大小。如果你想让不重要的信息变小，或者增大难以辨认的非拉丁字符的大小，这非常有用。你在此处所做的更改不会影响复习时卡片的显示方式：要执行此操作，请参阅模板部分。但是，如果你启用了"输入答案"功能，则你输入的文本将使用此处定义的字体大小。（有关键入答案时如何更改实际字体的信息，请参阅检查答案部分。）

**按此字段排序…** 告诉 Anki 在浏览器的"排序字段"列中显示此字段。你可以使用此功能按该字段对卡片进行排序。一次只能将一个字段设为排序字段。

**反转文本方向** 非常有用，如果你正在学习从右到左（RTL）显示文本的语言，例如阿拉伯语或希伯来语。此设置目前仅控制编辑；要确保复习期间文本正确显示，你需要调整模板。

**默认使用 HTML 编辑器** 如果你更喜欢直接在 HTML 中编辑字段，则此功能非常有用。

**默认折叠** 字段可以折叠/展开。动画可以在偏好设置中禁用。

**从不限定搜索中排除（速度较慢）** 如果你希望某个字段的内容不出现在不限定的（不限于特定字段的）搜索中，则可以使用此选项。

添加字段后，你可能希望将它们添加到卡片的正面或背面。有关更多信息，请参阅模板部分。

## 更改牌组/笔记类型

在添加过程中，你可以点击左上角的按钮更改笔记类型，点击右上角的按钮更改牌组。打开的窗口不仅允许你选择牌组或笔记类型，还可以添加新牌组或管理你的笔记类型。

## 组织内容

### 合理使用牌组

牌组旨在将你的内容划分为你希望分开学习的大类，例如英语、地理等等。你可能很想创建许多小牌组来组织你的内容，例如"我的地理书第 1 章"或"食物动词"，但**不推荐**这样做，原因如下：

- 许多小牌组可能意味着你最终会按可识别的顺序看到卡片。在旧版调度器中，新卡片只能按牌组顺序引入。而且，如果你打算依次点击每个牌组（这很慢），你最终会把所有"第 1 章"或"食物动词"的复习卡放在一起。这样更容易回答卡片，因为你可以从上下文中猜出答案，从而导致记忆不够牢固。当你需要在 Anki 之外回忆单词或短语时，你不会总能享受到先看到相关内容这种"奢侈"！
- 虽然在早期 Anki 版本中问题不太严重，但添加数百个牌组可能会导致卡顿，并且在 2.1.50 之前的 Anki 版本中，拥有数千个条目的超大牌组树实际上会破坏牌组列表的显示。

### 使用标签

与创建许多小牌组相比，更好的方法是使用**标签**和/或**字段**来对内容进行分类。标签是提升搜索结果、查找特定内容和保持卡片集井然有序的有效方法。使用标签和标记的方法有很多种，提前考虑你希望如何使用它们将帮助你决定哪种方法最有效。

有些人更喜欢使用牌组和子牌组来组织卡片，但与牌组相比，使用标签有一个很大的优势：你可以向单条笔记添加多个标签，但一张卡片只能属于一个牌组，这使得标签在大多数情况下比牌组更强大、更灵活的分类系统。你也可以像牌组一样以树形结构组织标签。

例如，你可以不创建"食物动词"牌组，而是将这些卡片添加到你的主语言学习牌组中，并使用"food"和"verb"标记这些卡片。由于每张卡片可以有多个标签，因此你可以执行搜索所有动词、搜索所有与食物相关的词汇，或搜索所有与食物相关的动词等操作。

你可以从编辑窗口和浏览器添加标签，也可以在那里添加、删除、重命名或整理标签。请注意，标签在笔记级别起作用，这意味着当你为一张带有兄弟卡片的卡片添加标签时，所有兄弟卡片也将被标记。如果你需要为单张卡片添加标签但不影响其兄弟卡片，则应考虑使用标记。

### 使用标记

**标记**（flags）与标签类似，但它们会在复习时显示在复习窗口中，在屏幕右上角显示一个彩色的标记图标。你还可以在浏览屏幕中搜索已标记的卡片，在浏览器中重命名标记，并从已标记的卡片创建筛选牌组。但与标签不同，一张卡片一次只能有一个标记。另一个重要的区别是标记在卡片级别起作用，因此对一张有兄弟卡片的卡片添加标记不会影响其兄弟卡片。

你可以在复习模式下（Windows 上按 `Ctrl+1-7` 或 Mac 上按 `Cmd+1-7`）以及浏览器中直接为卡片添加/取消标记。

### "已标记"标签

Anki 会特殊对待名为"marked"的标签。复习屏幕和浏览屏幕中有添加和删除"marked"标签的选项。当当前卡片的笔记具有该标签时，学习屏幕将显示一个星号。并且当笔记被标记时，其卡片在浏览屏幕中以不同的颜色显示。

> **注意**：标记功能主要是为了与旧版 Anki 保持兼容而保留的；大多数用户会希望改用标记（flags）。

### 使用字段

对于那些喜欢保持井井有条的人，你可以向笔记添加字段以对内容进行分类，例如"书"、"页"等。Anki 支持在特定字段中搜索，这意味着你可以执行`"book:my book" page:63`这样的搜索并立即找到你要查找的内容。

### 自定义学习和筛选牌组

使用自定义学习和筛选牌组，你可以从搜索条件创建临时牌组。这允许你大部分时间将内容混合在单个牌组中进行复习（以获得最佳记忆效果），同时在需要专注于特定材料时（例如考试前）创建临时牌组。一般规则是：如果你总是希望能够分别学习某些内容，则应将它们放在普通牌组中；如果你只是偶尔需要分别学习它们（考试、积压时等），则最好使用从标签、标记、星标或字段创建的筛选牌组。

## 编辑功能

编辑器在添加笔记、复习过程中编辑笔记或浏览时显示。

![编辑器图标](https://docs.ankiweb.net/media/editor_icons.png)

在左上角有两个按钮，它们分别打开字段和卡片窗口。

右侧是控制格式的按钮。粗体、斜体和下划线的工作方式与文字处理程序中的相同。接下来的两个按钮允许你创建下标或上标文本，这对于像 H2O 这样的化合物或像 x² 这样简单的数学方程式非常有用。然后还有两个按钮可以更改文本颜色。

**橡皮擦按钮** 清除当前所选文本中的所有格式——包括文本颜色、所选文本是否粗体等。接下来的三个按钮允许创建列表、文本对齐和文本缩进。

你可以使用**回形针按钮** 从计算机硬盘中选择音频、图像和视频并将其附加到笔记中。或者，你可以将媒体复制到计算机的剪贴板（例如，通过在网页上右键单击图像并选择"复制图像"），然后将其粘贴到要放置它的字段中。有关媒体的更多信息，请参阅媒体部分。

**麦克风图标** 允许你从计算机的麦克风录制并将其附加到笔记中。

**Fx 按钮** 显示向笔记添加 MathJax 或 LaTeX 的快捷方式。

**\[…] 按钮** 在选择完形填空笔记类型时可见。

**`< >`** **按钮** 允许编辑字段的基础 HTML。

![HTML 图标](https://docs.ankiweb.net/media/html_icon.png)

Anki 2.1.45+ 支持直接从编辑屏幕调整粘性字段。如果你点击字段右侧的**图钉图标**，Anki 在添加笔记后将不会清除该字段的内容。如果你发现自己正在将相同的内容输入到多条笔记中，你会发现这非常有用。在以前的 Anki 版本中，粘性字段是从字段屏幕切换的。

![图钉图标](https://docs.ankiweb.net/media/Pin_icon.png)

大多数按钮都有快捷键。你可以将鼠标光标悬停在按钮上以查看其快捷键。

粘贴文本时，Anki 默认会保留大多数格式。如果在粘贴时按住 `Shift` 键，Anki 将去除大部分格式。在"偏好设置"下，你可以切换"不带 Shift 键粘贴时去除格式"以修改默认行为。

## 完形填空删除

**完形填空删除**（Cloze deletion）是在句子中隐藏一个或多个单词的过程。例如，如果你有句子：

```
Canberra was founded in 1913.
```

……然后你对"1913"创建一个完形填空删除，那么该句子将变为：

```
Canberra was founded in [...].
```

有时以这种方式删除的部分被称为"被遮蔽"（occluded）。

有关为何要使用完形填空删除的更多信息，请参阅此处的规则 5。

Anki 提供了一种特殊的完形填空删除类型的笔记，以便轻松创建完形填空。要创建完形填空删除笔记，请选择"Cloze"笔记类型，然后在"Text"字段中输入一些文本。然后用鼠标拖动选择要隐藏的文本，并点击 **\[…]** 按钮。Anki 将把文本替换为：

```
Canberra was founded in {{c1::1913}}.
```

"c1"部分表示你在句子中创建了一个完形填空删除。如果需要，你可以创建多个删除。例如，如果你选择 Canberra 并再次点击 **\[…]**，文本将变为：

```
{{c2::Canberra}} was founded in {{c1::1913}}.
```

当你添加上述笔记时，Anki 将创建两张卡片。第一张卡片的问题将显示：

```
Canberra was founded in [...].
```

……并在答案上显示完整句子。另一张卡片的问题将显示：

```
[...] was founded in 1913.
```

你也可以在同一张卡片上同时省略多个部分。在上面的示例中，如果你将 c2 更改为 c1，则只会创建一张卡片，其中 Canberra 和 1913 都被隐藏。如果你在创建完形填空时按住 `Alt`（Mac 上为 `Option`），Anki 将自动使用相同的数字，而不是递增。

完形填空删除不需要落在单词边界上，因此如果你在上面的示例中选择"anberra"而不是"Canberra"，问题将显示为"C\[…] was founded in 1913"，为你提供一个提示。

你还可以给自己不与文本匹配的提示。如果你将原始句子替换为：

```
Canberra::city was founded in 1913
```

……然后在选择"Canberra::city"后按 **\[…]**，Anki 将把两个冒号后面的文本视为提示，将文本更改为：

```
{{c1::Canberra::city}} was founded in 1913
```

当卡片出现以供复习时，它将显示为：

```
[city] was founded in 1913.
```

有关测试你是否能够正确输入完形填空删除的能力的信息，请参阅键入答案部分。

从 2.1.56 版本开始，支持嵌套完形填空删除。例如，以下是有效的：

```
{{c1::Canberra was {{c2::founded}}}} in 1913
```

内部完形填空完全嵌套在外部完形填空中。不支持部分重叠，例如：

```
[...] founded in 1913 -> Canberra was
Canberra [...] in 1913 -> was founded
```

其中"was"一词同时出现在两个删除中。

当前实现只能处理有限数量的嵌套。在 Anki 24.11 中，它是 3 层。在其他版本中，限制约为 8 层，但当你接近限制时，Anki 可能会变慢。无法扩展此限制。如果你使用此功能，建议你将自己限制在几层嵌套之内。

在 2.1.56 版本之前，如果需要从重叠文本创建完形填空，请向你的完形填空添加另一个 Text 字段，将其添加到模板中，然后在创建笔记时，将文本粘贴到两个单独的字段中，如下所示：

```
Text1 字段: {{c1::Canberra was founded}} in 1913

Text2 字段: {{c2::Canberra}} was founded in 1913
```

默认的完形填空笔记类型有一个名为"Extra"的第二个字段，它显示在每张卡片的答案侧。它可用于添加一些用法说明或额外信息。

完形填空笔记类型被 Anki 特殊对待，不能基于常规笔记类型创建。如果你想自定义它，请确保克隆现有的"Cloze"类型而不是另一种笔记类型。可以自定义格式等内容，但无法向完形填空笔记类型添加额外的卡片模板。

## 图像遮蔽

Anki 23.10+ 原生支持图像遮蔽卡片。**图像遮蔽**（Image Occlusion，IO）笔记是基于图像而非文本的完形填空删除的一种特殊情况，允许你创建隐藏图像某些部分的卡片，测试你对隐藏信息的了解。

![图像遮蔽](https://docs.ankiweb.net/media/io.jpg)

### 添加图像

要将 IO 卡片添加到你的卡片集中，请打开"添加"屏幕，点击"类型"并从内置笔记类型列表中选择"图像遮蔽"。然后，点击"选择图像"以加载保存在计算机硬盘上的图像文件，或者如果你已将图像复制到剪贴板，则点击"从剪贴板粘贴图像"。

### 添加 IO 卡片

加载图像后，将打开 IO 编辑器。点击左侧的图标向图像添加任意数量的区域。有三种基本形状可供选择：

- 矩形
- 椭圆
- 多边形

你还可以为每个笔记选择两种不同的 IO 模式：

- **全部隐藏，猜一个**：所有区域都被隐藏，学习时一次只显示一个区域。
- **隐藏一个，猜一个**：一次只隐藏一个区域，学习时将显示该区域。其他区域将可见。

![图像遮蔽模式](https://docs.ankiweb.net/media/io_modes.jpg)

默认的 IO 笔记类型还具有标准字段：Header（显示在每张卡片正面和背面的图像上方）、Back Extra（显示在每张卡片背面的图像下方）和 Comments（不显示在卡片上）。要从 IO 编辑器访问这些字段，请点击"切换蒙版编辑器"按钮。在那里你还可以查看和编辑笔记的标签。

完成后，点击屏幕底部的"添加"按钮。Anki 将为你在上一步中添加的每个形状或形状组添加一张卡片，你可以正常开始复习。

## 编辑 IO 笔记

你可以在复习时点击"编辑"，或直接从浏览器编辑你的 IO 笔记。有几种你可以使用的工具。其中值得注意的有：

- **选择**：允许你选择一个或多个形状以移动、调整大小、删除或对它们进行分组。
- **缩放**：你可以使用鼠标滚轮自由移动图像并放大或缩小。
- **形状**（矩形、椭圆或多边形）：使用它们添加新的形状/卡片。
- **文本**：它将文本区域添加到你的图像中。这些文本区域可以移动、调整大小或删除，但使用此工具时不会创建卡片。
- **撤销/重做**。
- **放大/缩小 - 重置缩放**。
- **切换半透明**：使用此工具可临时查看隐藏区域。
- **删除**：使用此工具删除选定的形状和文本区域。请注意，删除形状不会自动删除其关联的卡片；你需要稍后使用"工具 > 清空卡片"，这与常规的完形填空删除相同。
- **复制**。
- **选择分组**：使用此工具创建形状的群集，这将允许你同时移动、调整大小或删除它们。请注意，两个或多个单形状分组后将只创建一张卡片。
- **取消分组**：选择一个组，然后单击此按钮以使每个形状再次独立。
- **对齐**：此工具可用于按需对齐你的形状/文本区域。

在复习 IO 卡片时，图像正下方将出现"切换蒙版"按钮。使用"全部隐藏，猜一个"模式时，此按钮将临时清除笔记的所有形状。

## 输入非拉丁字符和重音

所有现代计算机都内置了对输入重音和非拉丁字符的支持，并且有多种实现方式。我们推荐的方法是使用你要学习的语言的键盘布局。

具有独立文字的语言（如日语、中文、泰语等）具有特定于该语言的布局。

使用重音的欧洲语言可能具有自己的布局，但通常可以在通用的"国际键盘"布局上输入。它们的工作方式是先输入重音符号，然后再输入你想要重音的字符——例如，先输入撇号（´），然后输入字母 a（a），就得到 á。

## Unicode 规范化

像`á`这样的文本可以用多种方式在计算机上表示，例如使用该符号的特定代码，或使用标准的`a`，然后再加上一个用于顶部重音的代码。在混合来自不同来源的输入或使用不同计算机时，这会导致问题——如果你的计算机以一种形式处理键盘输入，但内容以另一种形式存储，则在搜索时将不匹配，即使最终结果看起来相同。

为确保内容可以在搜索中轻松找到，Anki 会将文本规范化为标准形式。对于大多数用户来说，此过程是透明的，但如果你正在学习某些材料（如过时的日语符号），规范化过程最终可能会将它们转换为更现代的等价物。

如果希望保留字符变体，调试控制台中的以下内容将关闭规范化：

```python
mw.col.conf["normalize_note_text"] = False
```

此后添加的任何内容都将保持不变。权衡之处在于，如果你在不同操作系统之间切换或从混合来源粘贴内容，你可能会发现很难搜索到内容。
