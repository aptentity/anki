## Week 1：入门与基础

### Day 1-2：认识与安装

**Day 1**
- [ ] 阅读配套文档《什么是 Anki》，了解核心概念（见前置准备）
- [ ] 安装 Anki 桌面端（使用中国镜像加速）
- [ ] 熟悉界面：牌组、卡片、统计、设置

**Day 2**
- [ ] 创建第一个牌组（如：英语单词）
- [ ] 创建 5-10 张基础卡片
- [ ] 完成首次复习体验
- [ ] 理解4个按钮含义：
  - Again = 重学（缩短间隔）
  - Hard = 稍短间隔
  - Good = 正常间隔
  - Easy = 稍长间隔

**📝 今日要点**
```
牌组（Deck）：存放卡片的容器
卡片（Card）：正面+背面的记忆单元
复习（Review）：根据算法出现的练习
```

### Day 3-4：卡片创建与模板

**Day 3**

- [ ] 创建带图片的卡片
- [ ] 创建带音频的卡片

**Day 4**

### Day 10-11：FSRS 算法与牌组设置

**Day 10**
- [ ] 了解 FSRS 算法原理（检索性、稳定性、难度）
- [ ] 对比 SM-2 与 FSRS 的区别：
  - SM-2：经典算法，固定公式计算间隔
  - FSRS：新一代算法，基于机器学习，更精准
- [ ] 在设置中启用 FSRS
- [ ] 观察 FSRS 对复习间隔的影响

**Day 11**
- [ ] 学习牌组选项配置
- [ ] 设置不同牌组的难度和复习量
- [ ] 理解"重新学习"与"恢复"机制

**📝 今日要点**
```
FSRS：新一代调度算法，比 SM-2 减少 20-30% 复习量
牌组选项：新卡片数、每日复习量、间隔Modifier
```

## ⚠️ 常见问题

**Q: 复习堆积怎么办？**
A: 使用"暂停"功能暂停不需要的牌组，或减少新卡片上限。

**Q: 卡片太多记不住？**
A: 遵循"少而精"原则，每张卡片聚焦一个知识点。

**Q: 中途想放弃？**
A: 坚持前两周是关键，养成习惯后就会轻松很多。
